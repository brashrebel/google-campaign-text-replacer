== Description ==

Want to dynamically insert keywords and phrases into your copy? This plugin comes bundled with several cool shortcodes that allow you to do just that.

To get value from this plugin you should be building URLs <a href="http://wordpress.org/plugins/google-url-creator/">using this plugin</a> and then sending traffic from various web sources to the new URLs.

== Installation ==

1. Upload to wp-content/plugins
1. Activate in Plugins
1. Go to Settings- - >Campaign Text
1. Have a ball!
